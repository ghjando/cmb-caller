[CMB_frontend_test_v2.py]

        證前端指令響應與連線狀態是否正常。
- 執行方式：

        python3 CMB_frontend_test_v2.py live
        
        # 執行測試環境測試 (Trial Mode)
        python3 CMB_frontend_test_v2.py trial
